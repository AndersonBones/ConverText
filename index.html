Bem Vindo ao Convert Text, uma ferramenta de Conversão de Texto.

Uma ferramenta de texto online muito útil onde você pode alternar entre letras minúsculas e maiúsculas, onde você pode capitalizar, minusculizar, misturar maiúsculas e minúsculas e transformar seu texto completamente.

Sobre as funções de conversão de texto

Sentence Case

O conversor de maiúsculas e minúsculas permitirá que você cole qualquer texto que desejar, e ele a transformará automaticamente em uma frase estruturada totalmente formada.

Funciona colocando em maiúscula a primeira letra de cada frase, e, em seguida, transformará o restante do texto em letras minúsculas, bem como converterá é em É. Cada letra após um ponto final será convertida em uma letra maiúscula.

Nota: no entanto, não colocará nomes ou lugares em maiúsculas.

Este é um exemplo de sentence case.

Lower Case

Se você está se perguntando como descapitalizar o texto, é exatamente isso que o conversor de texto em minúsculas permitirá que você faça - ele transforma todas as letras do seu texto em letras minúsculas.

este é um exemplo de lower case.

Upper Case

O transformador de maiúsculas pegará qualquer texto que você tenha e gerará todas as letras em maiúsculas. Ele essencialmente transformará todas as letras minúsculas em MAIÚSCULAS (assim como manterá as letras maiúsculas como letras maiúsculas).

ESTE É UM EXEMPLO DE UPPER CASE.

Capitalized Case

O conversor de maiúsculas converterá automaticamente a letra inicial de cada palavra em maiúscula e deixará as letras restantes como minúsculas.

Este É Um Exemplo De Capitalized Case.

Alternating Case

O conversor de maiúsculas e minúsculas permitirá que você transforme seu texto (não importa o formato atual) em texto que alterna entre letras minúsculas e maiúsculas. Ele irá gerar uma letra maiúscula e depois uma letra minúscula dentro da mesma palavra.

eStE É Um eXeMpLo dE aLtErNaTiNg cAsE.

Title Case

O conversor de maiúsculas e minúsculas é perfeito para aqueles que estão um pouco inseguros sobre como intitular um próximo ensaio. Essencialmente, garante que as letras corretas sejam maiúsculas no contexto de um título. Palavras como “um” serão deixadas todas em minúsculas e palavras importantes serão convertidas como “Título”.

Este é um Exemplo de Title Case.

Inverse Case

O conversor de maiúsculas e minúsculas permitirá que inverta de letras maiúsculas para minúsculas e vice-versa. Essencialmente, essa função converte cada letra de uma palavra para minúscula se ela inicialmente for maiúscula ou o contrário.

Bold Text

Uma maneira rápida de colocar seu texto em negrito. Basta digitar o texto normal no campo de texto e vê-lo automaticamente convertido para a versão em negrito.

Italic Text

Se você deseja colocar em itálico sua fonte, esta é uma ótima ferramenta. Digite seu texto normal e veja-o escrever automaticamente em uma inclinação, que você pode copiar e colar onde quiser.

Este é um Exemplo de Italic Text.

Underline text case

Sublinhe seu texto online com esta ferramenta de sublinhado. Escreva ou cole no conteúdo que você deseja sublinhar e, em seguida, você o verá gerado automaticamente sublinhado - que você pode copiar e colar onde quiser.

E̲s̲t̲é̲ ̲é̲ ̲u̲m̲ ̲E̲x̲e̲m̲p̲l̲o̲ ̲d̲e̲ ̲U̲n̲d̲e̲r̲l̲i̲n̲e̲ ̲T̲e̲x̲t̲

Mirror text case

Use este gerador de texto espelhado como uma maneira rápida e fácil de alterar o texto padrão para a versão espelhada. Basta digitar o texto normal e regular no painel esquerdo e vê-lo convertido para o texto espelhado no lado direito.

ƚxɘT ɿoɿɿiM ɘb olqmɘxɘ mu é ɘƚƨƎ

Wide Case

Uma ferramenta geradora de texto simples de usar que amplia e espaça seu texto. Ele basicamente converte seu texto normal em um texto com espaçamento maior entre as letras do texto.

Ｅｓｔｅ é ｕｍ Ｅｘｅｍｐｌｏ ｄｅ Ｗｉｄｅ Ｃａｓｅ

Strikethrough Text

Se você deseja uma maneira rápida de riscar seu texto, esta é uma ótima ferramenta. Digite o texto normal pelo qual deseja ser riscado e você o verá gerado automaticamente, pronto para copiar e colar.

E̶s̶t̶e̶ ̶é̶ ̶u̶m̶ ̶E̶x̶e̶m̶p̶l̶o̶ ̶d̶e̶ ̶S̶t̶r̶i̶k̶e̶t̶h̶r̶o̶u̶g̶h̶ ̶T̶e̶x̶t̶

Reverse Text

Se você deseja uma maneira rápida de fazer seu texto voltar para a frente, o gerador de texto reverso é ótimo. Escreva seu texto como normal e depois veja-o ser invertido.

txet esreveR ed olpmexE mu é etsE

Upside Down Text

Semelhante ao gerador de texto espelhado, bem como ao gerador de texto reverso, você pode virar seu texto de cabeça para baixo e de trás para frente com o Upside Down Text.

ʇxǝ┴ uʍoᗡ ǝpᴉsd∩ ǝp oldɯǝxƎ ɯn é ǝʇsƎ

Text to Morse code

Se você está procurando traduzir mensagens de código morse em portugues simples ou o contrário, este gerador de tradução online pode fazer exatamente isso. Ele também vem com um player de áudio para ouvi-lo.

-... . -- / ...- .. -. -.. ---

Text to Binary code

Traduza código binário para portugues e portugues para código binário com o seguinte gerador. Digite regularmente e obtenha uma série de 0 e 1 em troca.
